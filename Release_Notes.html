Purpose

STM32WB-BLE-sensors-multiple-links example demonstrates that the STM32WB can be used as a peripheral server and can handle up to 8 connections.
This example implements a multilink communication between a STM32WB5MM-DK board and up to 8 smartphones through Bluetooth® LE.
By creating simultaneous connections the board is able to share the same data with each smartphone.
Using the ST BLE Sensor app on the smartphones, it is possible to access the received data on different display screens.

STM32WB-BLE-sensors-multiple-links example is being delivered as-is, not necessarily supported by ST.

This example is based on the STM32CubeWB_v1.13.1 package. For further information about these components, see official package.
